# Contributing to SysGuard

Thank you for your interest in contributing to SysGuard!

## Getting Started
- Go 1.21 or later
- Docker (optional)
- Access to LLM API

## Development Setup
1. Clone the repository
2. Install dependencies: `go mod download`
3. Run tests: `make test`
4. Build: `make build`

## Code Style
- Follow standard Go formatting (`gofmt`)
- Use meaningful names
- Write clear comments

## Adding Skills
Implement the Skill interface and register in the registry.
